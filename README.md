# React TS Course Goals

A simple React + TypeScript app for tracking course goals. Add a goal with a short summary, see hints when the list is empty, and get a warning when you add many goals.

## Features
- Add and remove course goals
- Empty-state hint message
- Warning message after multiple goals
- Simple responsive UI

## Tech Stack
- React 19
- TypeScript
- Vite
